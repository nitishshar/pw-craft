About Us

pw-craft is a Playwright-first TypeScript library that packages sensible defaults, expressive helpers, and reporting you can hand to engineering leads and QA alike.

What you get Automation building blocks, not another bespoke framework
  • Browser lifecycle management with merged configuration from environment variables
  • Navigation, waiting, actions, assertions, storage, network, media, and accessibility utilities
  • Gherkin scenarios executed by Playwright via playwright-bdd, with traces and HTML reports
  • Self-contained HTML, JSON, and JUnit generators for custom pipelines

Reference documentation lives alongside the source in the main repository; this Angular app exists to exercise the toolkit against a realistic SPA.
